# Equinox
Equinox is a runtime for Roblox on Linux which uses a containerized approach to run a full Android system which can run the Roblox client. \
It is still a work-in-progress, and as such, has not been released as of yet. It will unfortunately not be open source at launch, but there are plans to make it open source eventually.

## Limitations you can expect at launch (which will be all be addressed)
- No ability to join via links from your browser
- No support for PC-exclusive games
- No Nvidia support
- No unprivileged container support (meaning you will have to give Equinox root access for now)
